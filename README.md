🚀 Key Features
- Interactive input system for activity-based emission data
- Real-time calculation of total carbon emissions
- Data visualization using bar and doughnut charts
- Comparative analysis (baseline vs optimized emissions)
- Responsive and user-friendly interface

🛠️ Technologies Used
- **HTML5** – Structured the application and defined core content
- **Tailwind CSS (CDN)** – Designed a responsive, dark-themed user interface with modern layouts
- **JavaScript (Vanilla)** – Implemented core logic, calculations, and dynamic UI interactions
- **Chart.js + DataLabels Plugin** – Visualized data through interactive bar and doughnut charts

- Designed and developed the application
- Implemented core features and logic
- Contributed to user experience and functionality
